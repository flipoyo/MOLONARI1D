Ce dossier contient un dossier pour les demos, un dossier pour la carte Arduino relay et un dossier pour la carte Arduino sensor. 
On n'utilise pas à l'heure actuelle les fichiers .ino mais les fichiers main.cpp (nécessaire dans l'utilisation de platformio). 
Un readme résume de façon consise les différentes librairies dans le dossier libs de firmware. 
Les fichiers conf.csv doivent être parfaitement à jour et mis sur les cartes SD des deux systèmes relays et sensors.
