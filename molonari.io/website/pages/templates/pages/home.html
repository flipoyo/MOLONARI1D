MOLONARI1D

Monitoring local stream-aquifer water and heat exchanges

An open-source ecosystem for building, deploying and operating underwater sensor networks that measure temperature and pressure in riverbeds.

Learn More View on GitHub

Core Components

🔧 Hardware

Arduino-based underwater sensors with LoRa communication, designed for 8–12 months of autonomous operation in riverbeds.

Explore hardware →

🔬 pyheatmy

Bayesian MCMC inference engine that estimates water and heat fluxes from sensor data with full uncertainty quantification.

Explore software →

📊 Molonaviz

PyQt5 desktop application for device management, data quality control and launching scientific analyses.
