Hardware

Arduino-based underwater sensor networks

Sensor Nodes

Each sensor node is built around an Arduino MKR WAN 1310 and includes temperature sensors (DS18B20), differential pressure sensors, an SD card for local storage and a LoRa antenna for communication.

  • Measurements every 15 minutes
  • Local CSV storage on SD card
  • Deep sleep between measurements (< 1 mA)
  • 8–12 months battery life on a 3.7 V 5000 mAh Li-ion cell

Relay Stations

Relay stations aggregate data from up to 10 sensor nodes via a custom LoRa protocol and forward it over LoRaWAN to the internet gateway.

  • Three-way handshake for reliable connections
  • Exponential backoff retry mechanism (up to 6 attempts)
  • Time-division multiple access to avoid collisions

Gateway

A Robustel R3000 gateway bridges LoRaWAN traffic to the internet via Ethernet, forwarding sensor data to the Django server as JSON payloads over HTTPS.

Deployment

Detailed assembly guides, waterproofing procedures and field deployment checklists are available in the hardware directory of the repository.
