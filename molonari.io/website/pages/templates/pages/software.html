Software

Python-based scientific computing ecosystem

pyheatmy – Scientific Computing Engine

pyheatmy implements Bayesian MCMC inference to estimate hydrological parameters from sensor data. It models a 1D riverbed column and provides:

  • Metropolis-Hastings sampling with adaptive step sizes
  • Uncertainty quantification through posterior distributions
  • Direct integration with Molonaviz databases and CSV exports
  • Multi-chain MCMC for improved convergence diagnostics

Requirements: Python 3.9+

pip install -e pyheatmy/

Molonaviz – Device Management & Visualisation

Molonaviz is a PyQt5 desktop application for operational monitoring. It manages the full laboratory hierarchy (sites → stations → sensors) and provides:

  • Device registration and configuration
  • Data quality control workflows
  • Real-time visualisation of sensor streams
  • Direct launching of pyheatmy inference

Requirements: Python 3.10+

pip install -e Molonaviz/

Quick Start

# Clone the repository
git clone https://github.com/flipoyo/MOLONARI1D.git
cd MOLONARI1D

# Install pyheatmy
pip install -e pyheatmy/

# Install Molonaviz dependencies
pip install pyqt5 pandas scipy matplotlib setuptools
